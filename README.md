## Hoe het werkt

### Plugins installeren
Download het `.plugin` bestand uit de plugin folder en open het in Claude Cowork. De plugin verschijnt als een installeerbare kaart — klik op accepteren en je bent klaar.

### Skills gebruiken
Skills worden automatisch geladen wanneer je een relevante vraag stelt. Ze bevatten domeinkennis die Claude helpt betere output te leveren.

## Filosofie

De meeste "AI tools" zijn losse prompts die generieke output produceren. Deze plugins zijn anders:

- **Orchestratie** — rollen bouwen voort op elkaars werk, niet in isolatie
- **Quality gates** — ingebouwde review die inconsistenties vindt en terugkoppelt
- **Jij blijft in control** — de AI stelt voor, jij beslist
- **Bewezen in productie** — getest op echte businesses, niet op demo's
